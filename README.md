# 📸 Creatibros | Landing Page & Portafolio 🎥

¡Bienvenido al repositorio oficial de **Creatibros**! Somos una agencia especializada en producción audiovisual, combinando la mejor calidad fotográfica con la magia del video moderno. ✨

---

## 🌟 ¿De qué trata este proyecto?

Esta plataforma es el hogar digital de Creatibros. Ha sido diseñada meticulosamente para mostrarle al mundo nuestro talento y conectar de forma directa con futuros clientes. A través de este sitio web, los usuarios pueden:

- 📷 **Navegar por increíbles galerías** de nuestras sesiones fotográficas de eventos (bodas, 15 años, cumpleaños, bautizos y primeras comuniones).
- 👔 **Descubrir nuestros servicios profesionales** orientados a impulsar marcas mediante fotografías gastronómicas, retratos corporativos, deportes y estilo boudoir.
- 🎬 **Explorar nuestro mundo Audiovisual**, con un fuerte enfoque en el poder del formato corto (Reels / TikToks) para redes sociales.
- 📲 **Contactarnos al instante** a través de un fácil acceso en toda la página (incluyendo un módulo para enviarnos mensajes directos por WhatsApp).

---

## 🚀 Características Principales

*   **⚡ Rendimiento Ultrarrápido:** Construida de cero utilizando Next.js 14 con el App Router y Server Components.
*   **🌓 Modo Oscuro (Dark Mode):** Un elegante diseño que hace una transición impecable entre temas claros y oscuros sin lastimar la legibilidad.
*   **📱 100% Responsiva:** Adaptada perfectamente para verse espectacular desde el teléfono celular hasta el monitor más grande.
*   **🎨 Interfaces inmersivas:** Estilos modulares alimentados por Tailwind CSS con paletas de colores únicas diseñadas para la marca de Creatibros (Morados eléctricos, lavandas y colores oscuros sofisticados).
*   **🔥 Carruseles Dinámicos:** Galerías de imágenes fluidas para una experiencia de usuario (UX) insuperable, creadas con Embla Carousel.

---

## 🛠️ Tecnologías Utilizadas

Este proyecto se ha desarrollado con algunas de las más recientes herramientas del ecosistema de React:

- **Framework:** [Next.js](https://nextjs.org/) (Versión 14+)
- **Diseño visual:** [Tailwind CSS](https://tailwindcss.com/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
- **Animaciones y Galerías:** Embla Carousel y Next-Themes

## 💻 Instalación y Uso Local

¿Quieres probar este proyecto en tu propia máquina? Sigue estos pasos:

1. **Clona este repositorio:**
   ```bash
   git clone https://github.com/juanandresariasutp/landing_CreatiBros.git
   ```

2. **Ingresa a la carpeta del proyecto:**
   ```bash
   cd "Landing Page Creatibros"
   ```

3. **Instala las dependencias necesarias:**
   ```bash
   npm install
   ```

4. **Inicia el servidor local:**
   ```bash
   npm run dev
   ```

5. **Prueba el sitio:**
   ¡Abre tu navegador! Entra a [http://localhost:3000](http://localhost:3000) y sorpréndete con los resultados. 🎉
